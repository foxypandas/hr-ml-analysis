# HR-анализ: удовлетворённость сотрудников и прогноз увольнения

Проект посвящён анализу HR-данных и решению двух прикладных задач машинного обучения:

1. прогнозирование уровня удовлетворённости сотрудника работой;
2. прогнозирование вероятности увольнения сотрудника.

В рамках проекта выполнены предобработка данных, исследовательский анализ, проверка статистических гипотез, обучение моделей машинного обучения и интерпретация результатов.

## Цель проекта

Цель проекта — выявить факторы, связанные с удовлетворённостью сотрудников и риском увольнения, а также построить модели, которые могут использоваться для поддержки HR-аналитики и принятия управленческих решений.

## Задачи проекта

### Задача 1. Прогноз удовлетворённости работой

Необходимо построить модель, предсказывающую уровень удовлетворённости сотрудника на основе его характеристик.

### Задача 2. Прогноз увольнения

Необходимо построить модель классификации, предсказывающую вероятность увольнения сотрудника, а также определить факторы, наиболее связанные с уходом сотрудников из компании.
